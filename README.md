# 🩺 Personalized Healthcare Recommendation System 
An ML-based healthcare recommendation system that predicts diseases based on user-reported symptoms and provides corresponding medicine recommendations.
This project demonstrates a complete end-to-end machine learning pipeline — from data preprocessing to model deployment with a web interface. 


## 🚀Features
- Symptom-based disease prediction using Machine Learning
- Personalized medicine recommendation
- Text preprocessing using NLP techniques
- Interactive web interface built with Streamlit
- Modular and scalable project structure
- Version-controlled with Git and GitHub 

## 🧠Tech Stack
- Programming Language: Python
- Machine Learning: Scikit-learn (Random Forest)
- Text Processing: CountVectorizer
- Web Framework: Streamlit
- Data Handling: Pandas, NumPy

## 📊Dataset Description
The dataset contains patient-related medical information such as:
- Symptoms – Text-based symptom descriptions
- Disease – Target variable for prediction
- Medicine – Recommended medication for the disease
Data preprocessing includes:
- Handling missing values
- Cleaning text data
- Removing irrelevant attributes

## 🧪Model Training & Evaluation
- Algorithm Used: Random Forest Classifier
- Problem Type: Multiclass Text Classification
- Evaluation Metrics:
    - Accuracy
    - Precision
    - Recall
    - F1-score
The model achieved an accuracy of ~63% on a limited dataset, which is acceptable for a proof-of-concept healthcare ML application.
